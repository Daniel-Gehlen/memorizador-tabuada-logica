Tautologias

Definição

"Tautologia é uma proposição sempre verdadeira, independente dos valores das partes que a compõem."

Macete Visual

Imagine um coringa do baralho (sempre serve, não perde nunca).

"Tautologia é o super-herói da lógica: nunca é derrotado pela falsidade!"

Exemplo Clássico: Princípio do Terceiro Excluído

Fórmula: p ∨ ¬p ("p OU não-p")

Por que é tautologia? Não há como p e ¬p serem falsos ao mesmo tempo. Um dos dois sempre será verdadeiro.

Técnica de Associação

Ligue tautologias a frases do cotidiano que são sempre verdadeiras:

  • "Ou chove ou não chove."
  • "Ou você está vivo ou não está."

Padrões para Identificar Tautologias

  • Padrão 1: Se a proposição contém (p ∨ ¬p), é tautologia.
  • Padrão 2: Se a tabela-verdade tem todas as linhas = V, é tautologia.

Exercício Prático

Verifique se é tautologia: (p → q) ∨ p

Contraste com Outros Conceitos

Conceito Definição Exemplo
Tautologia Sempre V p ∨ ¬p
Contradição Sempre F p ∧ ¬p
Contingência Pode ser V ou F p → q

O que NÃO é uma Proposição Lógica

Regra Geral

"Se não é possível atribuir Verdadeiro (V) ou Falso (F), não é proposição."

Exemplos de Frases que Não São Proposições

  • Perguntas: "Por que você está aqui?"
  • Ordens: "Feche a porta!"
  • Exclamações: "Que dia difícil!"
  • Paradoxos: "Esta frase é falsa."

Como Memorizar? Use o Acrônimo "P.O.V."

Letra Significado Exemplo
P Perguntas "Qual seu nome?"
O Ordens/Exclamações "Pare agora!" / "Uau!"
V Variáveis indefinidas "y > 3" (sem valor para y)

Dica Final

Proposições lógicas são como fatos verificáveis. Se não dá para "tirar foto" da verdade ou falsidade, não é proposição!

Exercício de Classificação

Classifique as seguintes frases como Proposição (P) ou Não-Proposição (N):

  • "2 + 2 = 4"
  • "Estude lógica!"
  • "x > 5" (sem valor para x)
  • "A neve é branca."

Macetes para Memorização

Negação (¬)

"Inverte o valor" - o que é V vira F e vice-versa

Conjunção (∧)

"E exige perfeição" - só é V quando ambos são V

Disjunção (∨)

"OU é inclusivo" - só é F quando ambos são F

Condicional (→)

"Vera Fischer é Falsa" - só é F quando V→F

Bicondicional (↔)

"Gosta de iguais" - só é V quando ambos valores são iguais
